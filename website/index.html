Forge

A simple GitHub client for fast operating around pull requests, issues, workflows, and repositories.
Forge is not a replacement for the GitHub website.
It keeps day-to-day work close and sends configuration back to GitHub.

Built for work, not configuration.

Forge keeps repositories, pull requests, workflow runs, checks, labels, and review actions in a dense three-pane layout. The GitHub website stays the source of truth for repository settings and advanced administration.

Fast paths stay close.

Search repositories, inspect current work, update pull request metadata, submit reviews, run workflows, and jump back to GitHub when the website is the right tool.
